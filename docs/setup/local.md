# Setup Local

## Requisitos

- Navegador moderno.
- Python 3 ou outro servidor estatico local.

## Execucao

Na raiz do projeto:

```powershell
npm run serve
```

Depois abra:

```text
http://localhost:4173
```

## Estrutura

- `index.html`: shell da aplicacao.
- `styles.css`: sistema visual responsivo.
- `src/domain`: modelo fisico.
- `src/application`: estado e particulas.
- `src/interfaces`: DOM e Canvas.
